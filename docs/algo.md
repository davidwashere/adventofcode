## Progressive Sum (Arithmetic Progression)

```
a1 = first num
aN = last num
n  = num of nums

n (a1 + aN)
-----------
     2
```

`1 + 2 + 3 + 4 + 5 = 15  (n=5, a1=1, aN=5)` --> `5(1+5)/2  =  30/2  =  15`

`2 + 3 + 4 + 5 = 14  (n=4, a1=2, aN=5)` --> `4(2+5)/2  =  28/2  =  14`

`2 + 5 + 8 + 11 = 26  (n=4, a1=2, aN=11)` ---> `4(2+11)/2 = 52/2 = 26`
